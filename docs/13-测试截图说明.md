## 截图方式

### 方式 1: 手动截图

1. 启动服务
   ```bash
   # 后端
   cd backend && npm start
   
   # 前端
   cd frontend && npm run dev
   ```

2. 访问页面
   - 首页：http://localhost:3000/
   - 列表：http://localhost:3000/houses
   - 详情：http://localhost:3000/houses/1
   - 登录：http://localhost:3000/auth
   - 用户：http://localhost:3000/user

3. 使用截图工具 (Shift+Cmd+4 或 PrintScreen)
