## 下载步骤

### 步骤 1: 访问图库网站

1. 打开 https://pexels.com/zh-cn/
2. 或打开 https://unsplash.com/

### 步骤 2: 搜索图片

使用上述表格中的搜索关键词

### 步骤 3: 下载图片

1. 选择喜欢的图片
2. 点击下载按钮
3. 选择合适尺寸

### 步骤 4: 重命名保存

按表格中的编号重命名：
```
mv downloaded_image.jpg assets/images/cover_01.jpg
```
