# 🎨 UI 设计规范

## 一、设计系统

### 1.1 UI 组件库

**使用**: UI-UX-PRO-MAX 组件库

```bash
npm install ui-ux-pro-max
```

#### 核心组件

| 组件 | 用途 | 示例 |
|------|------|------|
| `ProCard` | 房源卡片 | 房源展示 |
| `ProGallery` | 图片轮播 | 房源图片 |
| `ProFilter` | 高级筛选 | 房源筛选 |
| `ProContact` | 联系组件 | 联系销售 |
| `ProForm` | 表单组件 | 咨询表单 |
| `ProAvatar` | 头像组件 | 销售头像 |
| `ProBadge` | 徽章组件 | 房源状态 |

#### 使用示例

```vue
<template>
  <!-- 房源卡片 -->
  <ProCard
    :title="house.title"
    :price="house.price"
    :image="house.cover"
    :badges="['新上架', '热租']"
    @click="goToDetail"
  />
  
  <!-- 图片轮播 -->
  <ProGallery
    :images="house.images"
    :autoplay="true"
    :show-indicators="true"
  />
  
  <!-- 联系销售 -->
  <ProContact
    :agent="agent"
    :show-phone="true"
    :show-wechat="true"
    :show-chat="true"
  />
</template>
```

### 1.2 图标库

**使用**: IconPark (字节开源图标库)

```bash
npm install @icon-park/vue-next
```

#### 常用图标分类

| 分类 | 图标 | 用途 |
|------|------|------|
| 房屋 | `Home`, `Building`, `Door` | 房源相关 |
| 联系 | `Phone`, `Wechat`, `Chat` | 联系方式 |
| 用户 | `User`, `Users`, `Avatar` | 用户相关 |
| 位置 | `Location`, `Map`, `Position` | 位置相关 |
| 功能 | `Search`, `Filter`, `Sort` | 功能操作 |

#### 使用示例

```vue
<template>
  <div class="house-info">
    <IconPark name="home" :size="24" color="#333" />
    <span>{{ house.title }}</span>
  </div>
  
  <div class="contact-actions">
    <IconPark name="phone" @click="callAgent" />
    <IconPark name="wechat" @click="showWechat" />
    <IconPark name="chat" @click="startChat" />
  </div>
</template>

<script setup>
import { IconPark } from '@icon-park/vue-next'
</script>
```

## 二、设计规范

### 2.1 颜色规范

#### 主色调
```css
--primary: #1890ff;      /* 主色 */
--success: #52c41a;      /* 成功 */
--warning: #faad14;      /* 警告 */
--error: #f5222d;        /* 错误 */
```

#### 中性色
```css
--text-primary: #333333;    /* 主文字 */
--text-secondary: #666666;  /* 次要文字 */
--text-disabled: #999999;   /* 禁用文字 */
--border: #e8e8e8;          /* 边框 */
--background: #f5f5f5;      /* 背景 */
```

### 2.2 字体规范

```css
/* 字体大小 */
--font-xs: 12px;
--font-sm: 14px;
--font-md: 16px;
--font-lg: 18px;
--font-xl: 24px;
--font-xxl: 32px;

/* 字体粗细 */
--font-normal: 400;
--font-medium: 500;
--font-bold: 600;
```

### 2.3 间距规范

```css
--spacing-xs: 4px;
--spacing-sm: 8px;
--spacing-md: 16px;
--spacing-lg: 24px;
```

## 四、响应式设计

### 4.1 断点设置

```css
/* 手机 */
@media (max-width: 767px) { }

/* 平板 */
@media (min-width: 768px) and (max-width: 1023px) { }

/* 桌面 */
@media (min-width: 1024px) { }
```

### 4.2 适配规则

| 设备 | 布局 | 卡片数量 | 图片尺寸 |
|------|------|----------|----------|
| 手机 | 单列 | 1 列 | 100% |
| 平板 | 双列 | 2 列 | 50% |
| 桌面 | 多列 | 3-4 列 | 33%/25% |
